PlatformContext — единый стор

SyncManager

Монтируется один раз в App.tsx. При логине запускает все onSnapshot listeners, при логауте — отписывается. Управляет offline/reconnect состоянием.

PlatformContext

Единый useReducer стор. Хранит всё состояние платформы: user, enrollments, contracts, projects, notifications, stats, loading flags. Никто не ходит в Firestore напрямую.

Custom Hooks

Тонкие обёртки над Context. Компоненты используют только хуки — useEnrollments(), useContracts(), useRole(). Изолируют изменения структуры стора.

Shared Widgets

Чистые компоненты без логики — только рендер через props. StatCard, ContractCard, ProgressBar используются во всех дашбордах без дублирования.

RoleRouter

Читает user.roles из Context и монтирует нужный дашборд. Поддерживает комбинированные роли — ['student', 'lecturer'] показывает табы для обоих режимов.

# llm-toolkit

Comprehensive guide to 120+ LLM libraries for training, inference, RAG, agents, evaluation, and more.

## Overview

This skill organizes the LLM Engineer Toolkit into 15 categories covering the entire LLM development lifecycle:

1. **Training/Fine-Tuning** - Model training and fine-tuning libraries
2. **Application Development** - Frameworks and tools for building LLM apps
3. **RAG** - Retrieval-augmented generation tools
4. **Inference** - Local and production inference engines
5. **Serving** - Model serving frameworks
6. **Data Extraction** - Document/web data extraction
7. **Data Generation** - Synthetic data generation
8. **Agents** - Agent frameworks and tools
9. **Evaluation** - LLM benchmarking and evaluation
10. **Monitoring** - Observability and tracking
11. **Prompts** - Prompt engineering tools
12. **Structured Outputs** - Controlled generation
13. **Safety/Security** - LLM safety and security
14. **Embeddings** - Text embedding models
15. **Utilities** - Miscellaneous tools

## Source

Based on [llm-engineer-toolkit](https://github.com/KalyanKS-NLP/llm-engineer-toolkit) - a curated collection of 120+ LLM libraries.

## Quick Start

### Find libraries for a specific task

| Task | Recommended Libraries |
|------|----------------------|
| Fine-tune Llama | unsloth, Axolotl, PEFT |
| Local inference | llama.cpp, Ollama |
| Production serving | vLLM, LitServe |
| Build RAG app | LangChain, LlamaIndex |

| Monitor production | LangSmith, Opik, MLflow |

### Common workflows

**Local LLM setup:**
- Ollama or llama.cpp for running models
- llama.cpp for quantization
- Sentence-Transformers for embeddings

**Production LLM API:**
- vLLM for serving
- LiteLLM for multi-provider access
- RouteLLM for routing

**RAG pipeline:**
- LangChain or LlamaIndex for orchestration
- Chonkie for chunking
- Ragas for evaluation

**Agent development:**
- LangGraph or CrewAI for orchestration
- Composio for tools
- Memary for memory

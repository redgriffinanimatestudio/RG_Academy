# Run and deploy Red Griffin Creative Ecosystem

This contains everything you need to run your ecosystem locally.

View your app in AI Studio: https://ai.studio/apps/454c192e-a7ed-4d2f-8f2c-70278d1ea372

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## 🧪 Подход к тестированию с webapp-testing skill

### Используемые инструменты:
- **Playwright** - для автоматизации браузера
- **scripts/with_server.py** - для управления жизненным циклом серверов
- **Headless mode** - для запуска тестов без GUI

### Пример команды для запуска тестов:
```bash
python scripts/with_server.py \
  --server "cd api && npm run dev" --port 3000 \
  --server "npm run dev" --port 5173 \
  -- python test_endpoints.py
```

### Структура тестового скрипта:
```python
from playwright.sync_api import sync_playwright

with sync_playwright() as p:
    browser = p.chromium.launch(headless=True)
    page = browser.new_page()
    page.goto('http://localhost:5173')
    page.wait_for_load_state('networkidle')
    # ... тестирование эндпоинтов
    browser.close()
```

### Структура файлов для тестирования:
```
tests/
├── api_endpoints/
│   ├── test_analytics_sync.py
│   ├── test_finance_balance.py
│   ├── test_auth_logout.py
│   ├── test_notifications_mark_read.py
│   └── test_chat_unread_count.py
├── fixtures/
│   └── test_data.json
└── reports/
    └── test_results.html
```

### Команды для запуска тестов:
```bash
# Запуск всех тестов
python scripts/with_server.py \
  --server "cd api && npm run dev" --port 3000 \
  --server "npm run dev" --port 5173 \
  -- python -m pytest tests/api_endpoints/

# Запуск конкретного теста
python scripts/with_server.py \
  --server "cd api && npm run dev" --port 3000 \
  --server "npm run dev" --port 5173 \
  -- python tests/api_endpoints/test_analytics_sync.py

# Запуск с генерацией отчета
python scripts/with_server.py \
  --server "cd api && npm run dev" --port 3000 \
  --server "npm run dev" --port 5173 \
  -- python -m pytest tests/api_endpoints/ --html=reports/test_results.html
```
